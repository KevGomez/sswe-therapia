About This System

This scheduling system allows therapists to create available time slots for appointments, and clients to book those slots. The system prevents double bookings and allows for easy management of appointments.

  • Therapists can create 1-hour slots
  • Clients can view available slots by date
  • Clients can book and cancel appointments
  • All data is stored securely in Firebase
